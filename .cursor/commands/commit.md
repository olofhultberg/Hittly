Skapa ett commit-meddelande utifrån kodförändringen på detta format:
[feat/fix/style/chore/refactor]: [Beskrivning på svenska, en mening, max 100 tecken, första bokstaven ska vara stor, använd presens]

Exempel:
fix: Visa datum som tidzon Stockholm

Om kodförändringarna passar som en commit så skriv
✅ Passar som en commit
Annars skriv
⚠️ Rekommenderar att dela upp i flera commits p.g.a [...]

Fråga om att lägga till filer (git add <filnamn>) och sedan commit (git commit -m <commit message>) - om svar 'Ja' - Kör på, annars avbryt

Anropa inga verktyg.
